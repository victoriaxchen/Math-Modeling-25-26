CUSTOMIZE A SIMULATION:
-----------------------

from building import Building
from room import Room, RoomType, OccupantType
from responder import Responder, SweepStrategy, ExpertiseLevel
from simulation import SweepSimulation

# 1. Create a building
building = Building("My Building")

# 2. Add rooms
exit1 = Room("Exit1", area=5.0, room_type=RoomType.EXIT)
office1 = Room("Office1", area=30.0, room_type=RoomType.OFFICE, 
               occupant_type=OccupantType.NORMAL, has_smoke=False)

building.add_room(exit1)
building.add_room(office1)

# 3. Connect rooms with hallways
building.add_hallway("Exit1", "Office1", distance=10.0)

# 4. Create responders
firefighter = Responder(
    "FF1", "Firefighter 1",
    strategy=SweepStrategy.NEAREST_FIRST,
    expertise=ExpertiseLevel.EXPERT
)

# 5. Run simulation
simulation = SweepSimulation(building, [firefighter])
simulation.assign_starting_positions({"FF1": "Exit1"})
results = simulation.run_simulation()

# 6. View results
simulation.print_summary(results)

KEY PARAMETERS:
--------------

Room Types:
  - RoomType.OFFICE
  - RoomType.CONFERENCE
  - RoomType.STORAGE
  - RoomType.RESTROOM
  - RoomType.HALLWAY
  - RoomType.EXIT

Occupant Types:
  - OccupantType.NONE
  - OccupantType.NORMAL
  - OccupantType.LOW_MOBILITY
  - OccupantType.HIGH_DENSITY

Sweep Strategies:
  - SweepStrategy.NEAREST_FIRST (fastest for simple layouts)
  - SweepStrategy.SYSTEMATIC (consistent, predictable)
  - SweepStrategy.PRIORITY_BASED (best for smoke/occupancy variance)

Expertise Levels:
  - ExpertiseLevel.NOVICE (0.7x speed)
  - ExpertiseLevel.INTERMEDIATE (1.0x speed)
  - ExpertiseLevel.EXPERT (1.3x speed)
  - ExpertiseLevel.VETERAN (1.5x speed)

----------------------

results = {
